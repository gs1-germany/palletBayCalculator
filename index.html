GS1 Pallet Bay Calculator

a simple heuristic for calculating the number of required pallet bays for an order of trade items

useful for better planning, cost efficiency and sustainability

useful to inform your trading partners about the quantity of pallet bays via ORDERS, IFTMIN, and DESADV

For each order position, insert the following data:

Order Pos Trade item qty Original pallets Sandwich pallets Mixed pallets Sub-total
TOTAL 0 0 0 0 0
